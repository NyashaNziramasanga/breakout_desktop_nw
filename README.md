# Breakout! Game - Desktop

Desktop version of HTML/CSS/JS Breakout! game using the NW.js SDK (Node-Webkit)

Tutorial - https://youtu.be/5UsGnjPYxLU

## Usage

```bash
# Install nw.js, this takes a while
npm i nw@0.44.1-sdk nw-builder -D

# If installed run
npm install

# Run app
npm run dev

# Build app
npm run prod
```
